# Agent Memory Architectures

> [!summary]
> The structures that let agents retain and recall information beyond a single context window — short-term working memory, long-term stores, and episodic memory of past interactions.

**Domain:** [[tier-1-edge|AI & Agentic Architecture]]

## What it is

Agent memory architectures give agents persistence across turns and sessions. Short-term (working) memory holds the active task state; long-term memory stores durable facts and learned preferences, often in vector or structured stores; episodic memory records sequences of past events for recall and reflection. The architecture decides what is written, when it is retrieved, and how it is consolidated or forgotten.

## Key concepts

- Short-term / working memory
- Long-term memory stores
- Episodic memory and reflection
- Write/retrieve/consolidate policies
- Forgetting and memory hygiene
